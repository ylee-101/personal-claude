# TIL — 2026-04-02

## 맥 간 파일 복사 (SCP)

새 맥에서 **원격 로그인(SSH)**을 활성화해야 한다.

새 맥에서:
1. 시스템 설정 → 일반 → 공유
2. 원격 로그인 켜기

그 다음 기존 맥에서:
```bash
ssh <새맥유저명>@<새맥IP>
```

접속 확인되면 연결 끊고(`exit`) scp 명령어 실행.

```bash
scp -r ~/.claude/projects/ <새맥유저명>@<새맥IP>:~/.claude/projects/
```

### 참고
- 유저명 확인: `whoami`
- IP 확인: `ipconfig getifaddr en0`
